# VV Hi Framework

Public invention record for the framework layer of the `VV Hybrid Universe`.

## What This Repository Is

This repository exists to present the core invention layer separately from product UI, internal dashboards and ecosystem experiments.

It is the clean public home for:

- the framework declaration
- authorship and publication record
- the naming and structure of the core systems
- references back to live implementation inside the wider ecosystem

## Framework Scope

The `VV Hi Framework` currently groups:

- `VV HiS` - language
- `VV HiG` - identity
- `VV HiB` - architecture
- `VV HiA` - autonomy
- `VV HiL` - learning
- `VV HiC` - context
- `VV HiX` - exchange

## Positioning

- `VV Hybrid Universe` is the ecosystem
- `VV Nexus` is the system core
- `VV Hi Framework` is the invention layer
- `LEA`, `VNEX`, `VV Pulse`, and related systems are products built on top
